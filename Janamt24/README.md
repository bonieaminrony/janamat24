# জনমত ২৪ (Janamat 24)

জনমত ২৪ একটি বিশ্বস্ত এবং নির্ভরযোগ্য বাংলা সংবাদ মাধ্যম। সর্বশেষ জাতীয়, আন্তর্জাতিক, রাজনীতি, খেলাধুলা ও বিনোদন সংবাদ পরিবেশন করা আমাদের মূল লক্ষ্য।

## বৈশিষ্ট্যসমূহ (Features)

- **ডার্ক মোড (Dark Mode)**: প্রিমিয়াম ডার্ক থিম সাপোর্ট।
- **মেগা-গ্রিড লেআউট**: আধুনিক নিউজ পোর্টাল ডিজাইন।
- **লাইভ আপডেট**: ব্রেকিং নিউজ এবং লাইভ আপডেট টিকার।
- **মোবাইল রেসপন্সিভ**: সব ডিভাইসে নিখুঁত প্রদর্শন।

## লোকাল ডেভেলপমেন্ট (Local Development)

প্রজেক্টটি আপনার কম্পিউটারে চালানোর জন্য নিচের ধাপগুলো অনুসরণ করুন:

```sh
# ১. রিপোজিটরি ক্লোন করুন
git clone <YOUR_GIT_URL>

# ২. প্রজেক্ট ডিরেক্টরিতে প্রবেশ করুন
cd Janamt24

# ৩. ডিপেন্ডেন্সি ইনস্টল করুন
npm i

# ৪. ডেভেলপমেন্ট সার্ভার চালু করুন
npm run dev
```
